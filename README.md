# Airbnb India Analytics Dashboard

## Overview
This project is a full-stack data analytics dashboard built using Flask and Python. It provides interactive insights into Airbnb listings across India, including pricing trends, room types, ratings, and city-wise distribution.

## Features
- Interactive filters (City & Room Type)
- Dynamic KPI cards (Price, Rating, Listings)
- Dark mode dashboard UI
- India-focused map visualization
- Price trends and rating distribution charts
- Real-time data updates using Flask API

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- Data Processing: Pandas
- Visualization: Chart.js / Leaflet.js

## How to Run Locally

1. Clone the repository:
git clone https://github.com/aishidutta13/airbnb-dashboard.git

2. Navigate to the project folder:
   cd airbnb-dashboard
   
3. Install dependencies:
   pip install -r requirements.txt
   
4. Run the Flask app:
   python app.py
   
5. Open in browser:
   http://127.0.0.1:5000
   
## Key Insights
- Major cities dominate Airbnb listings in India
- Entire homes have higher average prices
- Ratings are mostly skewed towards higher values
- Price varies significantly across cities
