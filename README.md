# Airbnb India Analytics Dashboard

## Overview
This project is a full-stack data analytics dashboard built using Flask and Python. It provides interactive insights into Airbnb listings across India, including pricing trends, room types, ratings, and city-wise distribution.

## Features
- Interactive filters (City & Room Type)
- Dynamic KPI cards (Price, Rating, Listings)
- Dark mode dashboard UI
- India-focused map visualization
- Price trends and rating distribution charts
- Real-time data updates using Flask API

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- Data Processing: Pandas
- Visualization: Chart.js / Leaflet.js
